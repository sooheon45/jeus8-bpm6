A.1. 서버 Port

본 부록에서는 JEUS에서 사용하는 Port를 정리한다. 방화벽을 설정하는 경우 참고한다.

A.1. 서버 Port

  • BASEPORT

    설명 JEUS 서버가 JNDI 서비스 및 운영을 위해 필요한 기본 서비스 Port. base listener 설정으로 설정 가능
    Base Port9736

  • COS Naming Server Port

    설명 COS Naming 서비스를 위해 사용하는 Port
    Base PortBASEPORT + 4 (예: 9740)

  • ORB Port

    설명 IIOP Port
    Base PortBASEPORT + 1

  • ORB SSL Port

    설명 IIOP SSL Port
    Base PortBASEPORT + 2

  • ORB SSL Mutual Authorization Port

    설명 IIOP 상호인증 Port
    Base PortBASEPORT + 3

  • EJB RMI Port

    설명 EJB를 접근하기 위한 RMI Port
    Base PortBASEPORT + 7
