jeus.servlet.tcp
Class TCPDispatcherConfig 

java.lang.Object
  jeus.servlet.tcp.TCPDispatcherConfig
public class TCPDispatcherConfig
extends java.lang.Object
이 클래스는 Jeus WebContainer의 TCPListener를 통해 수신된 요청 메시지로부터 requestURI 정보를 얻어내기 위한 클래스이다. TCPServlet은 HttpServlet을 기반으로 하였기 때문에 TCPServlet을 호출하기 위해서는 HttpServlet을 호출할 때 필요한 requestURI가 필요하다.

HttpServlet을 호출하는 web browser는 URL 입력 박스를 통해 사용자가 직접 requestURI을 입력한다. TCPServlet의 경우는 이 클래스를 통해 requestURI를 구성하게된다.

requestURI는 contextPath + servletPath + pathInfo로 구성된다.

사용자가 구현한 TCPDispatcherConfig는 Jeus WebContainer가 부팅시 찾을 수 있는 곳 (예 : $JEUS_HOME/lib/application)에 jar형태로 보관되어 있어야 하며 domain.xml의 TCP 리스너 또는 Tmax 커넥터의 에 FQCN 으로 설정되어야 한다. (예: com.tmax.tcpconfig.TCPDispatcherConfigImpl)

isPersistentConnection

public boolean isPersistentConnection()
Determines whether TCP listner keeps client connection alive after finishing current request processing.

If return value is true, servlet engine will not disconnect the client connection after current request processing is finished. It might be useful for some proxy client.

This method only called once at booting time.

Even if this method returns true, client connection can be controlled through TCPServletResponse.closeOnExit.

Returns:
true if client connection need to keep alive
