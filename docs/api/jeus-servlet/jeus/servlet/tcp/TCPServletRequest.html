getBody

public byte[] getBody()
                    throws java.io.IOException
Deprecated.   

Returns the body of incoming request packet.

The length of the body is the result of TCPDispatcherConfig.getBodyLength. If it is equal to or less than zero, this method returns null. If it is a long, which means that it's bigger than Integer.MAX_VALUE, you cannot use this method.

If content length is too big, this method can cause OutOfMemoryError. Therefore, it is recommended to use getInputStream() and its read methods instead. But getReader() is not recommended because it is affected by request-encoding configuration.

Returns:
the request body
Throws:
java.io.IOException - if an I/O error occurs.
java.lang.UnsupportedOperationException - if the length of body is a long
java.lang.IllegalStateException - if ServletInputStream is already used
