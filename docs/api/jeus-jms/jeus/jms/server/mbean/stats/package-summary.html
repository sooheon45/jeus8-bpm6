Interface Summary

Package jeus.jms.server.mbean.stats Description

Java Message Service (JMS)에 관련된 통계 정보(statistic)에 대한 클래스들이 존재한다. 이 통계 정보에 대한 클래스들은 JSR77 Java EE Management 스펙에서 정의한 JMS 관련 MBean과 JEUS에서 정의한 MBean들이 이용한다.
