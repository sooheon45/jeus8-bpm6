MBean JMSDurableSubscriberResourceMBean

ObjectName Pattern:
JEUS:name={JMSDestinationResourceMBean-name}_JMSDur_{clientID}_{durableName}, j2eeType=JeusService,jeusType=JMSDurableSubscriberResource,*
Permission Name:

All Superinterfaces:
jeus.management.j2ee.J2EEManagedObjectMBean, jeus.management.j2ee.J2EEResourceMBean
MBean JMSDurableSubscriberResourceMBean
JMS Durable subscriber의 Resource를 나타내는 MBean이다.

이 MBean을 통해 해당하는 JMS Durable subscriber의 정보를 얻을 수 있다.

removeAllMessage

 int removeAllMessage()
Durable Subscription의 Queue에 있는 모든 메시지를 삭제한다.

Consume하기 위해 대기하고 있는 메시지와, Consume후 Acknowledge를 기다리고 있는 메시지들까지 모두 삭제한다.

Returns:
삭제된 메시지의 갯수
Throws:
javax.jms.JMSException - 메시지를 삭제할 권한이 없을 경우
