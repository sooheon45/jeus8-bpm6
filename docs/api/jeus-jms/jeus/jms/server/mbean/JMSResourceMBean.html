jeus.management.j2ee.J2EEManagedObjectMBean, jeus.management.j2ee.J2EEResourceMBean, jeus.management.j2ee.StatisticsProvider
MBean JMSResourceMBean
Java Message Service(JMS) 서버의 리소스를 나타내는 MBean이다.

이를 통해 해당하는 JMS 서버의 전체 정보를 얻을 수 있다.

삭제된 API

getJMSDestinationDigest(), getJMSSubscriptionDigest(), getJMSConnectionFactoryDigest() since JEUS 8

각각 Destinationm Subscription, ConnectionFactory에 대한 요약 정보를 가져온다. "요약된 정보"를 모아주는 별도의 기능은 없으며, 필요시 각각에 해당하는 MBean이나 console command를 통해 얻는다.

switchover() since JEUS 8

수동으로 강제로 switch-over하는 기능은 제공하지 않는다.
