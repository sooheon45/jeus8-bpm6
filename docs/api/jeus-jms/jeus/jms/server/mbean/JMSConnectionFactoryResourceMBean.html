ObjectName Pattern:

Permission Name:

JMSQueueConnectionFactoryResourceMBean, JMSTopicConnectionFactoryResourceMBean
MBean JMSConnectionFactoryResourceMBean
JMS의 ConnectionFactory의 Resource를 나타내는 MBean이다.

이 MBean을 통해 해당하는 JMS ConnectionFactory의 정보를 얻을 수 있다.
