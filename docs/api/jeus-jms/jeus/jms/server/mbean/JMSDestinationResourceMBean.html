MBean JMSDestinationResourceMBean

ObjectName Pattern:
JEUS:j2eeType=JeusService,jeusType=JMSDestinationResource,*
All Superinterfaces:
jeus.management.j2ee.J2EEManagedObjectMBean, jeus.management.j2ee.J2EEResourceMBean, jeus.management.j2ee.StatisticsProvider
All Known Subinterfaces:
JMSQueueDestinationResourceMBean, JMSTopicDestinationResourceMBean
MBean JMSDestinationResourceMBean

JMS Destination에 관한 작업을 하는 MBean이다.

이 MBean은 실제 MBean인 JMSQueueDestinationResourceMBean과 JMSTopicDestinationResourceMBean의 공통 부분을 나타낸다.
