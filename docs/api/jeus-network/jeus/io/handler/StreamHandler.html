public class StreamHandler
extends jeus.io.impl.nio.IdleTimeoutHolder
하나의 socket의 stream을 관리하는 객체를 나타낸다. 기본적으로 selector로부터 오는 event를 처리한다. application은 endpoint에 대해 socket 관리와 stream 처리 과정에서 생기는 여러가지 다음과 같은 작업들을 진행한다.

  • 자료 전송
  • socket close
  • exception reporting

크게 selector와 관련된 method, application interface에 해당하는 method로 구분된다.

StreamHandler는 Selector로 부터 읽기나 쓰기이벤트가 발생하면 이를 처리 해주는 역할을 하는 객체이다. 보통 이 객체는 어플리케이션 개발자가 직접 생성하거나 사용하는 경우는 없다. 다만 읽기를 담당하는 StreamContentReader나 쓰기를 담당하는 StreamContentWriter의 확장이 필요한 경우에는 StreamContentHandlerCreator 인터페이스를 구현하여 Endpoint생성시 설정을 하면 읽기나 쓰기의 기능을 확장할 수 있다.
