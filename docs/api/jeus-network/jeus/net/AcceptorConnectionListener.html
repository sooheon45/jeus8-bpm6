public interface AcceptorConnectionListener
extends MessageHandler
Acceptor를 사용하는 Endpoint에 대해 사용하는 ConnectionListener이다. ConnectionListener의 method 외에
      acceptor를 통해 들어오는 connection에 대한 event callback이 추가되어 있다.
      

      이 method들에서 공통으로 해야 할일은 다음과 같다.
      

      1. 주어진 packet write, 이를 callback method 내에서 강제하는 이유는 write 전후에 해야 할일이 있을수 있고
      packet에 대한 piggyback data를 추가할 시점도 각각 다르기 때문이다.
      

      2. connection 요청 packet에 piggyback되어 들어오는 data를 얻을 수 있다.
