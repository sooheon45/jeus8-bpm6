UDDIClient

public UDDIClient(java.net.URL inquiryURL,
                  java.net.URL publishURL)
UDDI Registry의 inquiry URL과 publish URL을 가지고 UDDIClient 객체를 생성한다.

Parameters:
inquiryURL - inquiry request들을 위해 사용되는 URL.
publishURL - publish request들을 위해 사용되는 URL.

UDDIClient

public UDDIClient(java.util.Properties config)
Configuration properties를 가지고 UDDIClient 객체를 생성한다.

Parameters:
config - UDDIClient를 위한 설정 정보를 담고 있는 Properties object. 이 properties object는 시스템 properties로 부터 정상적으로 얻은 세팅을 오버라이드 한다.

이 properties object에 의해 영향을 받는 현재 세팅은 다음과 같다:

jeus.uddi.client.inquiryURL
inquiry requests를 위해 UDDIClient에 의해 사용될 URL. setInquiryURL 메소드는 이 값을 오버라이드 한다.
jeus.uddi.client.publishURL
publish requests를 위해 UDDIClient에 의해 사용될 URL. setPublishURL 메소드는 이 값을 오버라이드 한다.
