Package jeus.workarea

javax.workarea 패키지는 JEUS에서 정의한 Work Area 서비스에 관련된 APIs를 담고 있다.

See:            Description

Interface Summary
UserWorkArea 사용자 Work Area에 접근하기 위한 인터페이스.
 

Enum Summary
PropertyModeType WorkArea에 추가되는 값의 Mode를 나타낸다.
 

Exception Summary
NotOriginatorException WorkArea가 Propagation되어 현재 사용되고 있는 부분이 실제 WorkArea를 시작한 곳이 아닐 경우 발생한다.
NoWorkAreaException WorkArea가 존재하지 않을 경우 발생한다.
PropertyReadOnlyException READ_ONLY로 설정된 값을 수정하거나 삭제하려 할 때 발생한다.
WorkAreaException WorkArea에서 발생하는 모든 Exception의 상위 Exception
