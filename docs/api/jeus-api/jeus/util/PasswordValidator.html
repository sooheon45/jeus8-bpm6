Interface PasswordValidator 


      
public interface PasswordValidator
사용자 암호를 설정할 때 암호의 적합성을 검사하기 위한 인터페이스.
      

      보안강화를 위하여 사용자의 암호를 간단하게 설정하는 것을 제한할 필요가 있을 경우,
      암호의 적합성 검사를 원하는 방식으로 진행할 수 있다.

JEUS 8

               Method Detail
            

         
validatePassword
public boolean validatePassword(java.lang.String id,
                                java.lang.String password)

            
사용자의 id와 password를 이용하여 주어진 password가 보안 기준에 적합한지를 검사한다.
               이 때 보안 기준을 정하여 해당 보안기준을 만족하면 true, 아니면 false를 반환한다.
            

            

            

               

                  
Parameters:
id - 적합성 검사를 진행할 사용자의 id
password - 적합성 검사를 진행할 사용자의 암호
                     

                  
                  
Returns:
주어진 암호의 적합성 여부, 기준에 적합하면 true, 아니면 false
