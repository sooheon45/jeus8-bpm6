Annotation Type CreateIdempotent 


         

      

      

      
@Target(value=TYPE)
@Retention(value=RUNTIME)
public @interface CreateIdempotent
Fail-Over 상황에서 Home create 호출이 idempotent 한지를 지정한다. 
      비즈니스(business) interface의 경우 빈 lookup 호출이 idempotent 한지를 지정한다. 
      

      Session 빈 클래스에 대해서 이 어노테이션을 지정할 수 있다.
      지정되지 않았을 경우 Stateless session 빈은 기본적으로 idempotent로 간주되며,
      Stateful session 빈은 idempotent 하지 않은 것으로 간주된다.
