Package jeus.schedule

JEUS Scheduler는 Scheduled task를 수행하는 서비스를 제공한다.

See:            Description

Interface Summary
ScheduleController Schedule 작업이 등록된 후 이를 제어하기 위한 Handle 객체.
ScheduleListener 이 interface는 Scheduler에 의해 스케쥴링될 작업(Task)을 나타낸다.
Scheduler Scheduler는 작업을 등록하는 인터페이스를 제공한다.
 

Class Summary
DailySchedule Deprecated.   
RemoteDailySchedule Deprecated.   
RemoteSchedule 원격으로 Schedule을 등록할 때 초기화 변수를 지정할 수 있는 Schedule 객체이다.
Schedule 이 class는 Scheduler에 의해 스케쥴링될 작업(Task)을 나타낸다.
SchedulerFactory SchedulerFactory는 Scheduler 객체를 얻는데 사용된다.
 

Exception Summary
JeusSchedulerException Scheduler에 관련된 문제가 생겼을 때 발생한다.
