public interface ScheduleListener
extends java.io.Serializable
이 interface는 Scheduler에 의해 스케쥴링될 작업(Task)을 나타낸다.
      

      이 interface는 Scheduler가 정해진 시간이 되었을 때 호출하는 callback method인 onTime()을 가지고 있다.
      사용자는 이 interface를 구현하여 주기적인 작업을 onTime()에 정의하고 Scheduler에
      등록하면 주어진 시간마다 호출된다.
