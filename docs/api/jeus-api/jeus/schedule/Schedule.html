Class Schedule

public class Schedule
extends java.lang.Object
implements java.io.Serializable, ScheduleListener
이 class는 Scheduler에 의해 스케쥴링될 작업(Task)을 나타낸다. 이 class는 JEUS 5.0 이전 Scheduler와의 호환성을 위해 제공된다. Scheduler에 의해 호출될 callback method로 onTime() 외에 nextTime()이 제공되며 nextTime()은 Scheduler에 등록시에 호출될 시간을 주지 않은 경우(5.0 이전방식)에 다음수행 시간을 결정하기 위해 호출된다.

작업을 정의하기 위해서는 이 클래스를 상속하여 onTime()과 nextTime() callback method를 override 한다.

Note: 모든 Schedule task는 multi-thread 방식으로 변경되었기 때문에 기존에 사용되던 threaded flag는 더 이상 사용되지 않는다.

nextTime

public long nextTime(long currentTime)
작업이 수행될 다음 절대 시간(milli second)을 리턴한다. Scheduler에 작업 등록시 registerSchedule()를 이용한 경우에만 사용된다. 이 메소드는 처음 onTime()이 불릴 시간을 알기위해 먼저 호출되며, onTime()이 호출된 후 다음 시간을 알기 위해서 호출된다. 이 method를 재정의하지 않으면 기본 값으로 현재 시간으로 부터 60초 이후를 넘겨준다.

Parameters:
currentTime - 현재 시간(millis)
Returns:
다음 작업 수행 시간(millis)
