public interface JDBCResourceServiceMBean
extends jeus.management.j2ee.J2EEManagedObjectMBean
JEUS 서버에 등록된 JDBC 리소스들을 관리하는 MBean이다.
      

      
이 MBean의 ObjectName의 요소

      
    
      
  • name key의 값 : 이 Service가 동작하는 JEUS Manager의 virtual host 이름
      
  • j2eeType key의 값 : JeusService
      
  • jeusType key의 값 : JDBCResourceService
      
  • isTargetable key의 유무 : 가지고 있지 않음
      
  • parent key list : J2EEServer
      
  • parent : J2EEServer
      
  • target : JeusServer (ignored)
      

      

      이 interface의 메소드 호출에 필요한 permission name은 jeus.server.node_name.container.container_name.JDBCResourceService.service_name
      이다.
      또한 필요한 action은 jdbcResource 이다.
