getRequestProcessingFlow
public java.util.List<java.lang.String> getRequestProcessingFlow(java.lang.String targetName,
                                                                 java.lang.String urlPattern,
                                                                 java.lang.String hostName)
                                                               throws jeus.util.JeusException

            
특정 서버 또는 클러스터에 디플로이된 웹 어플리케이션 중 입력한 URL 패턴과 호스트 정보에 맞는 서블릿 컨텍스트를 찾아서
               호출되는 필터와 서블릿의 흐름도를 보여준다.
            

            

            

               

                  
Parameters:
targetName - 서버 이름 또는 클러스터 이름
urlPattern - URL 패턴. "/" 로 시작하는 값이다. 컨텍스트 패스와 서블릿 또는 jsp 이름이 모두 있어야 한다.
hostName - 호스트 이름. DNS 또는 IP 주소. null일 경우 기본 가상 호스트으로부터 컨텍스트를 찾는다.
                     

                  
                  
Returns:
필터들과 서블릿 이름 리스트. 가장 끝에 있는 이름이 서블릿의 이름이다. 비어있을 경우 매핑된 서블릿이 없는 경우이다.

                  
                  
Throws:
jeus.util.JeusException - 컨텍스트를 찾을 수 없는 경우
java.lang.IllegalArgumentException - 필수 argument 가 null 일 경우, 알 수 없는 서버인 경우
