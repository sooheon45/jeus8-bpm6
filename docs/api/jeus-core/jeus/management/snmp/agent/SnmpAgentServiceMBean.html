Interface SnmpAgentServiceMBean 


         
All Superinterfaces:
jeus.management.j2ee.J2EEManagedObjectMBean

         
      

      

      
public interface SnmpAgentServiceMBean
extends jeus.management.j2ee.J2EEManagedObjectMBean
JEUS의 SNMP Service에 해당하는 MBean interface이다.
      

      
이 MBean의 ObjectName의 요소

      
    
      
  • name key의 값 : 이 Service가 동작하는 JEUS Manager의 virtual host 이름이나 Engine Container의 이름
      
  • j2eeType key의 값 : JeusService
      
  • isTargetable key의 유무 : false
      
  • parent key list: J2EEServer,
