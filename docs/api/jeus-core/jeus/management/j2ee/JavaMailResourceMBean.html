Property

    Read-Only Attribute

    Getter : 

     java.lang.String getProperty()
    설정 파일에 설정된 메일 세션(mail session) 정보.

    메일 세션의 프러퍼티의 키는 JavaMail 1.2 스펙을 따른다.

    • mail.user: 서버에서 인식하는 사용자 명
    • mail.host: 메일 서버의 IP 주소
    • mail.from: 보내는 측의 e-메일
    • mail.transport.protocol: 사용할 메일 프로토콜

    현재 JEUSMain.xml에 메일 세션 정보를 설정한다.

    Returns:
    메일 세션의 프러퍼티들의 키와 밸류
