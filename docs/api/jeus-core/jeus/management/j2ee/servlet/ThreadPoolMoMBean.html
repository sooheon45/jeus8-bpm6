jeus.management.j2ee.servlet
MBean ThreadPoolMoMBean

ObjectName Pattern:
JEUS:name={thread-pool-id},j2eeType=JeusService,jeusType=ThreadPool_WEBC,*
Statistics:
ThreadPoolStats
All Superinterfaces:
jeus.management.j2ee.J2EEManagedObjectMBean, jeus.management.j2ee.StateManageable, jeus.management.j2ee.StatisticsProvider
MBean ThreadPoolMoMBean
웹 컨테이너에서 이용하는 쓰레드 풀(Thread Pool)을 나타낸다.

웹 컨테이너에 존재하는 리스너로부터 받은 요청을 처리하는 작업 쓰레드(Worker Thread)의 풀을 의미한다. 쓰레드 풀에 대한 설정 정보를 제공하고, 각 설정을 제어하는 기능을 제공한다. 각 쓰레드에 대한 상세한 정보도 제공한다.

기존의 jeus.servlet.common.ThreadStateInfo 클래스가 jeus.management.j2ee.servlet.ThreadStateInfo 로 변경되었으므로 MBean 클라이언트를 재컴파일해야 한다.

interruptThread

 boolean interruptThread(long tid)
쓰레드 풀의 특정 쓰레드에 인터럽트(Interrupt)를 일으킨다. 즉, 쓰레드 풀에서 해당 쓰레드를 찾고, 그 쓰레드의 Thread.interrupt() 메소드를 호출한다. 쓰레드에 인터럽트를 일으킬 경우 플랫폼이나 운영체제에 따라 동작이 다를 수 있는 점을 유의한다.

Parameters:
tid - 인터럽트를 일으키고 싶은 쓰레드의 아이디(tid)
Returns:
해당 쓰레드가 존재할 경우 true, 그렇지 않으면 false
