Package jeus.management.j2ee.servlet

Java EE management (JSR77) 스펙에서 서블릿에 관련된 MBean과 JEUS에서 정의한 MBean들을 제공한다.

See:            Description

Interface Summary
SessionContainerCentralStats 중앙식 세션 클러스터링 사용시 세션 컨테이너의 연결 및 세션에 대한 통계정보를 나타내며, SessionContainerCentralMoMBean에서 이용한다.
SessionRouterStats 분산식 세션 클러스터링 사용시 세션 컨테이너의 연결 및 세션에 대한 통계정보를 나타내며, SessionRouterMoMBean에서 이용한다.
SessionStats 세션 컨테이너가 관리하는 세션에 대한 통계 정보를 나타내며, SessionContainerMoMBean에서 이용한다.
ThreadPoolStats 웹 컨테이너에서 이용하는 쓰레드 풀의 통계 정보를 나타내며, ThreadPoolMoMBean에서 이용한다.
WebEngineStats 웹 컨테이너의 JVM 메모리에 대한 통계 정보를 나타내며, WebEngineMoMBean에서 이용한다.
WebModuleStats 웹 모듈의 컨텍스트에 대한 요청 갯수, 처리 시간, 세션 정보등의 통계 정보를 나타내며, WebModuleMBean에서 이용한다.
 

MBean Summary
SessionContainerCentralMoMBean 중앙식 세션 클러스터링 사용시에 웹/EJB컨테이너에 존재하는 세션 컨테이너를 나타낸다.
SessionContainerMoMBean 웹/EJB컨테이너에 존재하는 세션 컨테이너를 나타낸다.
SessionRouterMoMBean 분산식 세션 클러스터링 사용시에 웹/EJB컨테이너에 존재하는 세션 컨테이너를 나타낸다.
ThreadPoolMoMBean 웹 컨테이너에서 이용하는 쓰레드 풀(Thread Pool)을 나타낸다.
WebEngineMoMBean 웹 컨테이너를 나타내며, 컨텍스트, 세션 서버, 쓰레드 풀에 대한 자원을 관리하고 제어한다.
WebListenerMoMBean 웹서버로부터 요청을 받는 리스너(Listener)를 나타낸다.
 

Class Summary
ThreadStateInfo 웹 컨테이너에서 이용하는 쓰레드 풀 내, 특정 쓰레드에 대한 정보를 나타낸다.
 

Package jeus.management.j2ee.servlet Description

Java EE management (JSR77) 스펙에서 서블릿에 관련된 MBean과 JEUS에서 정의한 MBean들을 제공한다. Java EE management 스펙의 메타 모델들을 MBean으로 구현하였다. 이외에도 MBean 인터페이스에서 부가적으로 사용하는 클래스들에 대해서 정의하였다.
