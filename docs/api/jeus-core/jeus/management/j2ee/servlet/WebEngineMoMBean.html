jeus.management.j2ee.servlet
MBean WebEngineMoMBean

ObjectName Pattern:
JEUS:name={engine-name},j2eeType=JeusService,jeusType=WebEngine,*
Statistics:
WebEngineStats
All Superinterfaces:
jeus.management.j2ee.J2EEManagedObjectMBean, jeus.management.j2ee.StatisticsProvider
MBean WebEngineMoMBean
웹 컨테이너를 나타내며, 컨텍스트, 세션 서버, 쓰레드 풀에 대한 자원을 관리하고 제어한다.

이 웹 컨테이너에 컨텍스트를 디플로이하고 언디플로이하는 기능을 제공하며, 세션 서버 정보, 쓰레드 풀에 대한 정보를 제공하며, 각 기능에 대해 제어하는 오퍼레이션들을 제공한다.
