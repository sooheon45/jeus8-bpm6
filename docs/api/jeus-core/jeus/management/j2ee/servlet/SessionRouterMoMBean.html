MBean SessionRouterMoMBean
분산식 세션 클러스터링 사용시에 웹/EJB컨테이너에 존재하는 세션 컨테이너를 나타낸다.

분산식 세션 서버(컨테이너)를 나타내며, 세션 서버의 설정관련 정보를 알려주고 설정값을 변경하는 기능을 제공한다.

분산식 세션 서버 설정으로 런타임에 connection-timeout, read-timeout, backup-trigger, 등의 정보를 알려주고, 런타임에 새로운 값으로 설정하는 기능을 제공한다.
