jeus.management.j2ee.servlet
Interface ThreadPoolStats 


         
All Superinterfaces:
javax.management.j2ee.statistics.Stats

         
      

      

      
public interface ThreadPoolStats
extends javax.management.j2ee.statistics.Stats
웹 컨테이너에서 이용하는 쓰레드 풀의 통계 정보를 나타내며,
       ThreadPoolMoMBean에서 이용한다.
      
      
쓰레드 풀에 현재 활성화된(active) 쓰레드의 개수, 블럭된(blocked) 쓰레드의 개수,
      처리를 기다리는 요청의 개수 등 다양한 정보를 제공한다.


         

            

            
See Also:
ThreadPoolMoMBean
