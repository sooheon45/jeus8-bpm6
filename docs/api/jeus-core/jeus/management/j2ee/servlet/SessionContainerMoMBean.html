jeus.management.j2ee.servlet
MBean SessionContainerMoMBean

ObjectName Pattern:
JEUS:name=engine-name-session,j2eeType=JeusService,jeusType=SessionContainer,*
Statistics:
SessionStats
All Superinterfaces:
jeus.management.j2ee.J2EEManagedObjectMBean, jeus.management.j2ee.StatisticsProvider
All Known Subinterfaces:
SessionContainerCentralMoMBean, SessionRouterMoMBean
MBean SessionContainerMoMBean
웹/EJB컨테이너에 존재하는 세션 컨테이너를 나타낸다.

세션 컨테이너가 관리하는 세션들의 key에 대한 정보를 얻거나, 특정 id의 세션의 존재 유무 확인 및 해당 세션을 invalidate하는 기능을 제공한다. 분산식의 경우 세션 서버와 세션 컨테이너는 동일한 객체를 가리킨다.

Session Container(세션 컨테이너): 웹 컨테이너나 EJB 컨테이너에서 세션 객체들을 관리하는 객체이다.
