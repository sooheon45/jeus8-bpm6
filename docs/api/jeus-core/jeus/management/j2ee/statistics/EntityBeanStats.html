getAutoRemovedEJBObjectCount
public javax.management.j2ee.statistics.CountStatistic getAutoRemovedEJBObjectCount()

            
명시적으로 remove를 호출해서가 아닌, 설정한 disconnect-timout 시간이 지나 타임아웃으로 삭제된 빈 인스턴스 개수.
               
               
 엔터티 빈의 경우 사용자가 remove를 호출 하지 않아도 disconnect-timeout이 지나면 자동으로 빈 인스턴스를 삭제해 준다.

            

            

            

               

                  
Returns:
타임아웃으로 삭제된 빈 인스턴스 개수.
