Package jeus.management.j2ee.statistics

Java EE management (JSR77) 스펙에서 정의한 통계 정보(Statistic)에 대한 인터페이스와 구현체들이 존재한다.

See:            Description

Interface Summary
EJBStats EJB 컴포넌트에 대한 통계 정보를 나타내며, 하위 구체적인 EJB에 대한 통계 정보의 공통 부분을 나타낸다.
EntityBeanStats EJB 엔터티 빈에 대한 통계 정보를 나타내며, EntityBeanMBean에서 이용한다.
JTAStats 트랜잭션에 대한 통계 정보를 제공하며, JTAResourceMBean에서 이용한다.
JVMStats JVM의 통계 정보를 나타내며, JVMMBean에서 이용한다.
MessageDrivenBeanStats Message-Driven 빈에 대한 통계 정보를 나타내며, MessageDrivenBeanMBean에서 이용한다.
SessionBeanStats 세션 빈에 대한 통계 정보를 나타낸다.
SingletonSessionBeanStats User: miju Date: 2009.
StatefulSessionBeanStats Stateful 세션 빈에 대한 통계 정보를 제공하며, StatefulSessionBeanMBean에서 이용한다.
StatelessSessionBeanStats Stateless 세션 빈에 대한 통계 정보를 제공하며, StatelessSessionBeanMBean에서 이용한다.
