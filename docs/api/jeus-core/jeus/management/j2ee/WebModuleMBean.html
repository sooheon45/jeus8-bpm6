MBean WebModuleMBean
서버에 디플로이 되어있는 웹모듈을 나타낸다.

이 웹모듈이 가지고 있는 서블릿 컴포넌트들에 대한 정보 제공 및 이 웹모듈의 컨텍스트에 대한 제어와 각 서블릿에 대한 제어 기능을 제공한다.
